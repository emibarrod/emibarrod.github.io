MoMIR

Cuenta automatizada de Twitter. El propósito es irónico: desmotivar a los estudiantes de medicina que opositan al examen MIR. Esta idea surgió de una broma y la acabé llevando a cabo. Cosas de ingenieros supongo. La cuenta es @motivacionmir.

Tecnologías usadas:

  • Google Cloud
  • Python
  • Twitter API
  • OpenAI API
  • GPT-3
